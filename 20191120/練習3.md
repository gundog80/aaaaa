# 練習
製作丙級第二題健康天地頁的標籤面板  

## 提示
- `.show()` 可以顯示東西
- `.hide()` 可以隱藏東西
- `.index()` 可以知道某東西在同層是第幾個

## 版面
```html
<div id="btns">
    <input type="button" value="健康新知">
    <input type="button" value="菸害防制">
    <input type="button" value="癌症防治">
    <input type="button" value="慢性病防治">
</div>
<div id="content">
    <div>健康新知</div>
    <div>菸害防制</div>
    <div>癌症防治</div>
    <div>慢性病防治</div>
</div>
```
